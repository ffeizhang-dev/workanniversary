## 🎨 头像生成方法

### 方法1：使用在线工具生成

#### 推荐工具

1. **Canva（最简单）**
   - 网址：https://www.canva.cn
   - 搜索"机器人头像"或"吉祥物"
   - 选择模板，改成青色系
   - 添加周年相关元素（日历、蛋糕等）
   - 免费下载PNG格式

2. **Notion Avatar Maker**
   - 网址：https://notion-avatar.vercel.app
   - 生成可爱的卡通头像
   - 选择青绿色系配色
   - 添加配饰（帽子、围巾等）

3. **Avataaars Generator**
   - 网址：https://getavataaars.com
   - 自定义卡通人物
   - 调整颜色为理想色系
   - 下载SVG或PNG

4. **Flaticon / Iconfinder**
   - 搜索：anniversary bot、calendar、celebration
   - 下载免费图标
   - 使用Canva组合编辑

## 🎨 头像快速制作教程

### 使用Canva制作（5分钟）

#### 步骤：

1. **访问Canva**
   ```
   https://www.canva.cn
   注册/登录（可用微信）
   ```

2. **创建设计**
   ```
   点击"创建设计"
   选择"Logo" 或 "社交媒体图标"
   尺寸：400x400 像素
   ```

3. **选择模板**
   ```
   搜索关键词：
   - "mascot"（吉祥物）
   - "robot"（机器人）
   - "calendar"（日历）
   - "anniversary"（周年）
   ```

4. **自定义设计**
   ```
   改变颜色：
   - 主色：#00796B（青绿色）
   - 辅色：#FF9520（橙色）

   添加元素：
   - 点击"元素" → 搜索"calendar"
   - 添加礼物盒、星星等装饰
   ```

5. **导出下载**
   ```
   点击右上角"分享"
   选择"下载"
   格式：PNG，透明背景
   ```

6. **上传到飞书**
   ```
   飞书群 → 添加机器人
   上传刚才下载的图片
   ```
