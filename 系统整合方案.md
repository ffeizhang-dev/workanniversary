## 🎨 导航样式代码

在每个页面的 `<style>` 标签中添加：

```css
/* 导航菜单 */
.nav-menu {
    display: flex;
    gap: 8px;
    align-items: center;
}

.nav-link {
    display: flex;
    align-items: center;
    gap: 6px;
    padding: 10px 20px;
    border-radius: 8px;
    color: #4E5969;
    text-decoration: none;
    font-size: 15px;
    font-weight: 500;
    transition: all 0.2s;
    background: transparent;
}

.nav-link:hover {
    background: rgba(0, 121, 107, 0.08);
    color: #00796B;
}

.nav-link.active {
    background: linear-gradient(135deg, #00796B 0%, #00695C 100%);
    color: white;
    box-shadow: 0 2px 8px rgba(0, 121, 107, 0.2);
}

/* 移动端适配 */
@media (max-width: 768px) {
    .nav-menu {
        flex-direction: column;
        gap: 4px;
        width: 100%;
    }

    .nav-link {
        width: 100%;
        justify-content: center;
    }
}
```

## 📊 数据共享

三个页面共享同一个 localStorage，数据结构保持一致：

### 共享数据
```javascript
// 员工数据
localStorage.getItem('employees')

// 飞书配置
localStorage.getItem('feishuConfig')

// 祝福记录（可选）
localStorage.getItem('blessingHistory')
```

### 数据流转
```
员工管理页面
    ↓
添加/编辑员工数据
    ↓
保存到 localStorage
    ↓
祝福确认页面自动读取
    ↓
检查今日周年员工
    ↓
生成祝福语
    ↓
读取飞书配置
    ↓
发送到飞书
```

## 🆘 常见问题

### Q1: 导航栏显示不正常？

**A**: 检查CSS样式是否正确添加，确保 `.nav-menu` 和 `.nav-link` 样式都存在。

### Q2: 点击导航跳转404？

**A**: 检查文件名是否正确，确保三个HTML文件都已上传到GitHub。

### Q3: 数据不同步？

**A**: localStorage是基于域名的，确保三个页面在同一个域名下访问（都是 ffeizhang-dev.github.io/workanniversary/）。

### Q4: 移动端导航挤在一起？

**A**: 检查是否添加了响应式CSS（@media查询）。
