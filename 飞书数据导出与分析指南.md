# 📊 飞书群评估数据导出与分析指南

## 🎯 概述

所有评估数据都通过飞书机器人发送到您的私密收集群，每条评估是一个消息卡片。本指南教您如何导出、整理和分析这些数据。

---

## 方法1：飞书桌面端导出聊天记录（推荐）

### 适用场景
- 需要永久保存评估数据
- 需要离线查看和分析
- 评估数量较多（>20份）

### 操作步骤

#### 第1步：打开飞书桌面客户端

1. **启动飞书桌面版**（不是手机端）
   - Windows: 桌面图标或开始菜单
   - Mac: 应用程序文件夹

2. **进入私密收集群**
   - 找到【私密】员工评估数据收集群
   - 点击进入

#### 第2步：导出聊天记录

1. **点击群聊右上角** "⋯" （三个点）

2. **选择"更多"** → **"导出聊天记录"**

3. **配置导出设置**：
   ```
   时间范围：选择评估活动的开始日期到结束日期
   例如：2024-01-01 到 2024-01-31

   导出格式：
   - TXT（纯文本，推荐用于数据处理）
   - PDF（格式化文档，推荐用于存档）

   内容选项：
   ☑ 文本消息
   ☐ 图片（不需要勾选）
   ☐ 文件（不需要勾选）
   ```

4. **点击"导出"**
   - 选择保存位置（建议创建专门的文件夹）
   - 文件名示例：`评估数据_2024年1月.txt`

5. **等待导出完成**
   - 导出时间取决于消息数量
   - 20份评估约需要10-30秒

#### 第3步：查看导出的文件

打开导出的TXT文件，内容格式如下：

```

📊 收到新的匿名评估

被评估人: 张三
部门: 技术部
职位: 技术总监
评估编号: ANON-abc123-1234567890
提交时间: 2024-01-15 14:30:25

📈 评分详情
战略目标的承接者: 5 分
经营的责任者: 4 分
产品交付的责任者: 5 分
技术创新的引领者: 5 分
组织的领导者: 4 分
跨部门协作的统筹者: 5 分

总分: 28/30 分 (93.3%)
评级: 卓越非凡

---

[2024-01-15 15:20:18] 评估数据收集机器人
📊 收到新的匿名评估
...
```

## 方法2：手动整理到Excel（适合少量数据）

### 步骤1：创建Excel表格

1. **打开Excel**，创建新工作簿

2. **创建表头**（第1行）：
   ```
   A列: 评估编号
   B列: 被评估人
   C列: 部门
   D列: 职位
   E列: 维度1-战略目标
   F列: 维度2-经营责任
   G列: 维度3-产品交付
   H列: 维度4-技术创新
   I列: 维度5-组织领导
   J列: 维度6-跨部门协作
   K列: 总分
   L列: 百分比
   M列: 评级
   N列: 提交时间
   O列: 备注/建议
   ```

3. **美化表头**：
   - 加粗字体
   - 添加背景色（例如浅蓝色）
   - 设置边框

### 步骤2：从飞书群复制数据

1. **打开飞书群**，找到每条评估消息

2. **复制评估数据**：
   - 长按消息 → 复制
   - 或直接查看消息内容

3. **粘贴到Excel对应列**：
   ```
   被评估人: 张三 → 粘贴到B列
   部门: 技术部 → 粘贴到C列
   维度1: 5分 → 输入 5 到E列
   ...依此类推
   ```

4. **重复以上步骤**，处理所有评估消息

### 步骤3：添加Excel公式

在Excel中添加自动计算：

```excel
// K2单元格（总分）- 如果是手动输入的分数
=SUM(E2:J2)

// L2单元格（百分比）
=K2/30*100

// M2单元格（评级）- 使用IF函数
=IF(L2>=85,"卓越非凡",IF(L2>=70,"表现优秀",IF(L2>=50,"符合预期","需要改进")))

// 平均分（在数据下方）
=AVERAGE(E2:E100)  // 维度1的平均分
=AVERAGE(K2:K100)  // 总分的平均分
```

### 步骤4：创建数据透视表

1. **选择所有数据**（包括表头）

2. **插入** → **数据透视表**

3. **配置数据透视表**：
   ```
   行标签：部门
   值：
   - 平均总分
   - 计数（评估人数）
   - 各维度平均分
   ```

4. **生成报表**：
   - 按部门统计
   - 按评级分布
   - 按维度分析

## 方法3：使用Python脚本自动提取（高级）

### 适用场景
- 评估数量很多（>50份）
- 需要批量处理
- 有一定技术基础

### Python脚本示例

```python
import re
import pandas as pd
from datetime import datetime

def parse_feishu_export(txt_file):
    """解析飞书导出的TXT文件"""

    with open(txt_file, 'r', encoding='utf-8') as f:
        content = f.read()

    # 按评估分割
    assessments = content.split('📊 收到新的匿名评估')

    data = []

    for assessment in assessments[1:]:  # 跳过第一个空白
        try:
            # 提取信息
            name = re.search(r'被评估人:\s*(.+)', assessment).group(1).strip()
            dept = re.search(r'部门:\s*(.+)', assessment).group(1).strip()
            position = re.search(r'职位:\s*(.+)', assessment).group(1).strip()
            assess_id = re.search(r'评估编号:\s*(.+)', assessment).group(1).strip()
            submit_time = re.search(r'提交时间:\s*(.+)', assessment).group(1).strip()

            # 提取6个维度分数
            scores = re.findall(r':\s*(\d+)\s*分', assessment)
            if len(scores) >= 6:
                dim1, dim2, dim3, dim4, dim5, dim6 = [int(s) for s in scores[:6]]

            # 提取总分和评级
            total_match = re.search(r'总分:\s*(\d+)/30', assessment)
            total_score = int(total_match.group(1)) if total_match else sum([dim1,dim2,dim3,dim4,dim5,dim6])

            rating_match = re.search(r'评级:\s*(.+)', assessment)
            rating = rating_match.group(1).strip() if rating_match else ''

            # 提取备注（如果有）
            comment = ''
            comment_match = re.search(r'【备注/改进建议】\n(.+)', assessment, re.DOTALL)
            if comment_match:
                comment = comment_match.group(1).strip()

            data.append({
                '评估编号': assess_id,
                '被评估人': name,
                '部门': dept,
                '职位': position,
                '战略目标': dim1,
                '经营责任': dim2,
                '产品交付': dim3,
                '技术创新': dim4,
                '组织领导': dim5,
                '跨部门协作': dim6,
                '总分': total_score,
                '百分比': round(total_score/30*100, 1),
                '评级': rating,
                '提交时间': submit_time,
                '备注': comment
            })
        except Exception as e:
            print(f"解析错误: {e}")
            continue

    return pd.DataFrame(data)

# 使用方法
df = parse_feishu_export('评估数据_2024年1月.txt')

# 导出到Excel
df.to_excel('评估数据汇总.xlsx', index=False)

print(f"✅ 已成功提取 {len(df)} 份评估数据")
print(f"📊 平均总分: {df['总分'].mean():.1f}")
print(f"📈 评级分布:\n{df['评级'].value_counts()}")
```

### 如何运行脚本

1. **安装Python**（如果没有）
   - 下载：https://python.org/downloads
   - 安装时勾选"Add to PATH"

2. **安装必要的库**
   ```bash
   pip install pandas openpyxl
   ```

3. **保存脚本**
   - 将上面的代码保存为 `parse_feishu.py`
   - 放在和TXT文件同一个文件夹

4. **运行脚本**
   ```bash
   python parse_feishu.py
   ```

5. **查看结果**
   - 自动生成 `评估数据汇总.xlsx`
   - 包含所有评估数据的结构化表格

## 📊 数据分析方法

### 1. 基础统计分析

#### 在Excel中计算：

```excel
// 提交率
=COUNTA(B2:B100)/总人数*100

// 平均总分
=AVERAGE(K2:K100)

// 标准差（离散度）
=STDEV(K2:K100)

// 各维度平均分
=AVERAGE(E2:E100)  // 维度1
=AVERAGE(F2:F100)  // 维度2
...依此类推
```

#### 创建统计表格：

| 统计项 | 数值 | 公式 |
|--------|------|------|
| 总评估数 | 32 | =COUNTA(B2:B100) |
| 平均总分 | 24.5 | =AVERAGE(K2:K100) |
| 最高分 | 30 | =MAX(K2:K100) |
| 最低分 | 15 | =MIN(K2:K100) |
| 卓越非凡 | 8人 | =COUNTIF(M2:M100,"卓越非凡") |
| 表现优秀 | 18人 | =COUNTIF(M2:M100,"表现优秀") |
| 符合预期 | 5人 | =COUNTIF(M2:M100,"符合预期") |
| 需要改进 | 1人 | =COUNTIF(M2:M100,"需要改进") |

### 2. 维度分析

创建维度对比表：

| 维度 | 平均分 | 占比 | 排名 |
|------|--------|------|------|
| 战略目标 | 4.2 | 84% | 3 |
| 经营责任 | 3.8 | 76% | 5 |
| 产品交付 | 4.5 | 90% | 1 |
| 技术创新 | 4.3 | 86% | 2 |
| 组织领导 | 3.9 | 78% | 4 |
| 跨部门协作 | 3.7 | 74% | 6 |

**Excel公式**：
```excel
// 平均分
=AVERAGE(E2:E100)

// 占比
=E2/5*100

// 排名
=RANK(平均分, 所有维度平均分范围, 0)
```

### 3. 部门对比分析

使用数据透视表：

1. **选择数据** → **插入** → **数据透视表**

3. **生成部门对比表**：

| 部门 | 评估人数 | 平均总分 | 平均百分比 |

## ⚠️ 数据安全注意事项

1. **导出文件保护**
   - 为Excel文件设置密码
   - 存储在安全位置（公司内网）
   - 限制访问权限

2. **敏感信息处理**
   - 评估编号是随机的，无法追溯个人
   - 管理层报告中隐藏敏感细节
   - 个人报告单独发送，不公开

3. **数据保留期**
   - 原始数据保留1年
   - 统计报告永久保存
   - 定期清理不需要的副本

4. **访问权限**
   - 只有HR和管理层可以查看原始数据
   - 其他人员只能看到统计结果
   - 被评估人只能看到自己的报告

## 📞 技术支持

如果遇到问题：

1. **飞书导出失败**
   - 确认使用桌面版（不是网页版）
   - 检查时间范围是否正确
   - 尝试缩短时间范围重新导出

2. **数据格式错误**
   - 使用UTF-8编码打开TXT文件
   - 确保完整导出（不要中途中断）

3. **Excel公式不工作**
   - 检查单元格引用是否正确
   - 确认数据类型（数字不要存为文本）
