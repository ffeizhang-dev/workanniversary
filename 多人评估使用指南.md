# 📊 多人在线评估完整指南

## 🎯 方案概述

本系统支持多人匿名评估，采用**分布式数据收集 + 集中合并导出**的方案。

### 核心优势
- ✅ **无需服务器**：完全本地运行，零成本
- ✅ **完全匿名**：每个人独立填写，互不影响
- ✅ **数据安全**：数据在本地，不上传云端
- ✅ **灵活导出**：支持CSV、Excel、JSON多种格式

---

## 📋 完整流程（3个角色）

### 角色1：组织者/HR（发起人）

#### 步骤1：准备文件
将以下3个文件打包发送给所有评估人：
- `assessment-survey.html` - 评估问卷
- `使用说明.txt` - 填写指南（见下方）
- （可选）被评估人名单

#### 步骤2：发送通知
通过邮件/飞书/企业微信发送：

```
【重要】匿名能力评估问卷填写通知

各位同事：

请大家配合完成XXX的匿名能力评估，具体步骤：

1. 下载附件中的 assessment-survey.html
2. 双击打开文件（用浏览器打开）
3. 填写评估内容并提交
4. 点击"💾 导出我的数据"按钮
5. 将导出的JSON文件发送给我

截止时间：2025年XX月XX日
保证完全匿名，请大家客观真实填写

感谢配合！
```

#### 步骤3：收集数据
收集所有评估人发送的JSON文件，保存到一个文件夹

#### 步骤4：合并导出
1. 打开 `assessment-collect.html`
2. 将所有JSON文件拖拽上传
3. 点击"📥 导出为 Excel"
4. 得到完整的评估数据汇总表

---

### 角色2：评估人（填写者）

#### 步骤1：打开问卷
双击 `assessment-survey.html` 文件，浏览器会自动打开

#### 步骤2：填写评估
1. **查看匿名说明**
   - 页面顶部会显示"🔒 本问卷完全匿名"
   - 系统自动生成匿名评估编号
   - 不会记录您的任何身份信息

2. **填写被评估人信息**
   - 姓名：被评估人的姓名
   - 部门：被评估人所在部门
   - 职位：（可选）被评估人职位

3. **进行评分**
   - 6个评估维度，每个维度0-5分
   - 点击对应的分数按钮
   - 右侧会实时显示总分

4. **提交评估**
   - 点击"✅ 提交评估"按钮
   - 看到成功提示

#### 步骤3：导出数据
点击"💾 导出我的数据"按钮，会自动下载一个JSON文件

文件名示例：`我的评估数据_1708857234567.json`

#### 步骤4：发送文件
将下载的JSON文件通过邮件/微信/飞书发送给组织者

#### 可选：评估多人
如果需要评估多个人：
1. 提交当前评估后
2. 点击"🔄 重置"按钮
3. 重新填写下一个人的评估
4. 全部完成后，再点击"💾 导出我的数据"
5. 所有评估会在一个JSON文件中

---

### 角色3：分析者（查看结果）

#### 方式1：使用收集工具查看

1. **打开收集工具**
   ```
   双击 assessment-collect.html
   ```

2. **上传所有JSON文件**
   - 点击上传区域或拖拽文件
   - 支持同时选择多个文件
   - 系统会自动合并数据

3. **查看统计信息**
   - 📊 总评估数
   - 📈 平均得分
   - 👥 被评估人数

4. **导出分析报告**
   - **CSV格式**：适合Excel打开
   - **Excel格式**：带中文标题，直接可用
   - **JSON格式**：用于二次开发

#### 方式2：使用结果查看器

1. **加载数据**
   - 在收集工具中点击"👁️ 查看汇总结果"
   - 或直接打开 `assessment-results.html`

2. **查看可视化分析**
   - 📊 统计卡片：总评估数、平均分、卓越数
   - 📈 维度分析：6个维度的平均得分柱状图
   - 📋 详细表格：所有评估记录

3. **筛选和排序**
   - 按被评估人筛选
   - 按总分排序
   - 按提交时间排序

## 🔧 常见问题

### Q1：评估人能看到别人的评分吗？
**A**：不能。每个人的数据存储在自己的浏览器中，互不影响。

### Q2：如何保证匿名性？
**A**：
- 系统只生成随机评估编号
- 不记录IP地址、设备信息
- 不记录评估者姓名
- JSON文件中只有评估内容

### Q3：评估人误删了JSON文件怎么办？
**A**：
1. 重新打开 `assessment-survey.html`
2. 浏览器中的数据还在
3. 再次点击"导出我的数据"

### Q4：可以修改已提交的评估吗？
**A**：
- 在提交前可以随时修改
- 提交后无法修改（保证数据一致性）
- 如需修改，清除浏览器数据后重新填写

### Q5：收集工具能处理多少数据？
**A**：
- 理论上无限制
- 实测1000条记录无压力
- 建议单次不超过5000条

### Q6：导出的Excel乱码怎么办？
**A**：
- 使用"导出为 Excel"按钮（带BOM）
- 不要用"导出为 CSV"
- 或用记事本打开CSV，另存为UTF-8编码

### Q7：能否在手机上填写？
**A**：
可以，但建议用电脑：
- 手机：可填写，导出需要技术操作
- 电脑：完整功能，一键导出

### Q8：数据安全吗？
**A**：
- ✅ 完全本地存储
- ✅ 不上传任何服务器
- ✅ 不联网运行
- ✅ 可随时删除

### Q9：可以导出PDF吗？
**A**：
可以，点击"📥 导出PDF"按钮，实际上是调用浏览器打印功能：
1. 点击按钮
2. 在打印对话框选择"另存为PDF"
3. 保存文件

### Q10：如何清空所有数据？
**A**：
- 评估人：浏览器按 F12 → Console → 输入 `localStorage.clear()`
- 管理员：在结果查看器中点击"清空数据"按钮

```json
[
  {
    "assessmentId": "ANON-7F8A2B-KLMN123",
    "employeeName": "张三",
    "department": "技术部",
    "position": "技术总监",
    "assessmentDate": "2025-02-25",
    "isAnonymous": true,
    "scores": {
      "dimension1": 5,
      "dimension2": 4,
      "dimension3": 5,
      "dimension4": 4,
      "dimension5": 5,
      "dimension6": 4
    },
    "totalScore": 27,
    "submittedAt": "2025-02-25T10:30:00.000Z"
  }
]
```

### 导出Excel表头

| 列名 | 说明 |
|------|------|
| 评估编号 | 匿名ID |
| 被评估人 | 姓名 |
| 部门 | 所属部门 |
| 职位 | 职位信息 |
| 维度1 | 战略目标的承接者（0-5分）|
| 维度2 | 经营的责任者（0-5分）|
| 维度3 | 产品交付的责任者（0-5分）|
| 维度4 | 技术创新的引领者（0-5分）|
| 维度5 | 组织的领导者（0-5分）|
| 维度6 | 跨部门协作的统筹者（0-5分）|
| 总分 | 0-30分 |
| 评级 | 卓越非凡/表现优秀/符合预期/需要改进 |
| 提交时间 | 评估提交的时间戳 |

## 🎓 高级技巧

### 技巧1：批量评估模板

创建一个Excel模板，列出所有被评估人：
```
姓名    部门      职位
张三    技术部    技术总监
李四    产品部    产品经理
...
```

评估人可以对照模板逐一填写。

### 技巧2：数据去重

如果同一个评估人提交了多次，可以在Excel中：
1. 按"评估编号"排序
2. 识别重复的编号
3. 保留最新的一条记录

### 技巧3：统计分析

在Excel中可以使用数据透视表：
- 按被评估人分组，查看平均分
- 按维度分组，查看薄弱环节
- 按评级分组，查看分布情况

### 技巧4：自动化收集

使用脚本批量处理JSON文件：
```bash
# 合并所有JSON文件（Linux/Mac）
jq -s 'add' *.json > merged.json
```
