## 🚀 快速配置（5分钟）

### 方式1：飞书群机器人（最简单，推荐）

#### 第1步：创建飞书群

1. 打开飞书
2. 创建新群：`【周年祝福通知群】`
3. 拉入HR团队成员

#### 第2步：添加机器人

1. 群设置 → 群机器人 → 添加机器人
2. 选择"自定义机器人"
3. 机器人名称：`入职周年提醒`
4. 安全设置：选择"关键词"，输入：`周年`
5. **复制Webhook地址**：
   ```
   https://open.feishu.cn/open-apis/bot/v2/hook/xxxxxxxx
   ```

#### 第3步：在系统中配置

1. **打开飞书配置页面**：
   ```
   双击打开：feishu-config.html
   或访问：https://你的GitHub用户名.github.io/仓库名/feishu-config.html
   ```

2. **选择"Webhook机器人"模式**

3. **粘贴Webhook地址**

4. **点击"测试连接"**
   - 如果飞书群收到测试消息，说明配置成功！

5. **点击"保存配置"**

#### 第4步：配置自动提醒

1. 在主系统中设置：
   - 提醒时间：每天上午9:00
   - 提醒方式：飞书群聊
   - 提前天数：当天

2. 启用自动提醒

#### 完成！

每天早上9点，系统会自动检查当天入职周年的员工，并发送祝福消息到飞书群。

## 🔧 方式2：飞书开放平台API（私聊员工）

### 适用场景
- 需要私聊员工发送祝福
- 需要更高的消息发送频率
- 需要交互式功能

### 配置步骤（需要管理员权限）

#### 第1步：创建企业自建应用

1. 访问：https://open.feishu.cn/
2. 登录企业管理员账号
3. 开发者后台 → 企业自建应用
4. 创建应用：`入职周年祝福系统`

#### 第2步：申请权限

在应用的"权限管理"中申请：
- ✅ `im:message` - 发送消息
- ✅ `im:message:send_as_bot` - 以机器人身份发送
- ✅ `contact:user.base:readonly` - 获取用户信息

#### 第3步：获取凭证

在"凭证与基础信息"页面：
- **App ID**: `cli_xxxxxx`
- **App Secret**: `xxxxxx`（保密）

#### 第4步：在系统中配置

1. 打开 `feishu-config.html`
2. 选择"开放平台API"模式
3. 填写 App ID 和 App Secret
4. 测试连接
5. 保存配置

#### 第5步：获取员工飞书User ID

在员工信息中添加"飞书User ID"字段：
- 方法1：飞书管理后台查询
- 方法2：让员工发送消息给机器人自动获取

#### 完成！

系统会直接私聊员工发送周年祝福。

## 🎨 自定义祝福模板

系统支持自定义祝福内容，可以使用变量：

```
🎉 {name}入职{years}周年纪念

亲爱的{name}，

恭喜您加入{company}大家庭{years}周年！

您在{department}的出色表现，
体现了{position}的专业素养。

从{joinDate}到今天，
感谢您{years}年的付出与坚守！

期待与您共创更美好的未来！🚀

────────────────────────
{company} · 入职周年祝福系统
{currentDate}
```

**支持的变量**：
- `{name}` - 员工姓名
- `{years}` - 入职年数
- `{department}` - 部门
- `{position}` - 职位
- `{joinDate}` - 入职日期
- `{company}` - 公司名称
- `{currentDate}` - 当前日期

## ⚠️ 常见问题

### Q1: 测试连接失败？

**检查清单**：
- [ ] Webhook URL是否正确复制
- [ ] 飞书群中机器人是否还在
- [ ] 安全设置中关键词是否配置
- [ ] 网络连接是否正常

**解决方法**：
1. 重新复制Webhook URL
2. 确认机器人未被移除
3. 检查关键词设置（如：周年）

### Q2: 群里收不到消息？

**可能原因**：
- 消息未包含关键词
- 机器人被禁言
- 消息格式错误

**解决方法**：
1. 确保消息包含关键词"周年"
2. 检查机器人状态
3. 查看系统错误日志

### Q3: 能否同时发送到多个群？

**答案**：可以！

**方法**：
1. 创建多个机器人
2. 在系统中配置多个Webhook
3. 选择发送时勾选所有群

### Q4: 消息发送频率限制？

**飞书限制**：
- Webhook：20条/分钟
- API：根据企业版本不同

**系统处理**：
- 自动限速（1秒/条）
- 分批发送大量消息
- 显示发送进度
