📦 多人评估数据收集工具

收集多人填写的评估数据，合并导出统一结果

📚 使用流程

1

分发评估问卷

将 assessment-survey.html 文件发送给所有评估人

2

评估人填写并导出

每位评估人完成问卷后，点击浏览器控制台运行导出命令

// 在问卷页面按 F12 打开控制台，粘贴以下代码并回车： const data = JSON.parse(localStorage.getItem('anonymousAssessments') || '[]'); const blob = new Blob([JSON.stringify(data, null, 2)], {type: 'application/json'}); const url = URL.createObjectURL(blob); const a = document.createElement('a'); a.href = url; a.download = '我的评估数据_' + Date.now() + '.json'; a.click();
3

收集JSON文件

让每位评估人将导出的JSON文件发送给您（邮件/微信/飞书等）

4

上传并合并数据

在下方上传所有收集到的JSON文件，系统会自动合并
