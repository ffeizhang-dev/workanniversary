# 🚀 启动本地服务器指南

## 问题说明

从本地文件（file://）访问飞书API会遇到CORS跨域问题。解决方法是通过HTTP服务器访问。

---

## 方法1：使用Python启动服务器（最简单）

### 如果已安装Python：

**打开命令行**，进入项目文件夹，运行：

```bash
# Python 3.x
python -m http.server 8000

# 或者 Python 2.x
python -m SimpleHTTPServer 8000
```

然后访问：
```
http://localhost:8000/assessment-survey.html
http://localhost:8000/feishu-test.html
```

---

## 方法2：使用Node.js启动服务器

### 如果已安装Node.js：

```bash
# 安装 http-server
npm install -g http-server

# 在项目文件夹运行
http-server -p 8000
```

然后访问：`http://localhost:8000`

---

## 方法3：使用VSCode Live Server

如果使用VSCode编辑器：

1. 安装扩展："Live Server"
2. 右键HTML文件 → "Open with Live Server"
3. 自动在浏览器中打开

---

## 方法4：使用Chrome禁用CORS（仅测试用）

**⚠️ 警告：这会降低浏览器安全性，仅用于测试，测试完后关闭！**

### Windows:
```cmd
"C:\Program Files\Google\Chrome\Application\chrome.exe" --disable-web-security --user-data-dir="C:\temp\chrome_dev"
```

### Mac:
```bash
open -na "Google Chrome" --args --disable-web-security --user-data-dir="/tmp/chrome_dev"
```

### Linux:
```bash
google-chrome --disable-web-security --user-data-dir="/tmp/chrome_dev"
```
