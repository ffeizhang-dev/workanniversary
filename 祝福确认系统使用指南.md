### 第2步：准备员工数据

系统会从浏览器localStorage读取员工数据，格式如下：

```javascript
// localStorage中的员工数据示例
const employees = [
    {
        id: "1",
        name: "张三",
        department: "技术部",
        position: "技术总监",
        joinDate: "2021-01-15",  // 入职日期
        reminderEnabled: true     // 是否启用提醒
    },
    {
        id: "2",
        name: "李四",
        department: "产品部",
        position: "产品经理",
        joinDate: "2023-01-15",
        reminderEnabled: true
    }
];

// 保存到localStorage
localStorage.setItem('employees', JSON.stringify(employees));
```

**如果您已有入职周年系统**，员工数据应该已经存在，无需额外设置。

---

### 第3步：配置飞书Webhook

系统会读取之前配置的飞书Webhook：

```javascript
// localStorage中的飞书配置
{
    "mode": "webhook",
    "webhookUrl": "https://open.feishu.cn/open-apis/bot/v2/hook/xxx",
    "enabled": true
}
```

**如果还没配置**，请先访问 `feishu-config.html` 配置Webhook。

## ❓ 常见问题

### Q1: 打开页面显示"今天没有员工周年"？

**A**: 检查：
- localStorage中是否有员工数据
- 员工的入职日期是否正确
- 员工的 `reminderEnabled` 是否为 true

### Q2: 点击"立即发送"没反应？

**A**: 检查：
- 是否配置了飞书Webhook
- 打开浏览器控制台查看错误信息
- Webhook地址是否正确

### Q3: 生成的祝福语信息不完整？

**A**: 检查员工数据是否包含：
- name（姓名）
- department（部门）
- position（职位）
- joinDate（入职日期）

### Q4: 能否预览飞书消息效果？

**A**: 可以先发送一条测试消息：
1. 添加测试员工
2. 生成祝福语
3. 发送到测试群查看效果

### Q5: 如何添加更多模板？

**A**: 修改HTML文件中的 `templates` 对象，添加新模板即可。
