# 📊 评估数据导出指南

## 🎯 数据存储位置

根据您的配置，评估数据存储在：
- **飞书私密收集群** - 每份评估会自动发送为一条消息卡片

---

## 方法1：直接查看飞书群消息（最简单）

### 适用场景
- 少量评估（<20份）
- 只需要查看，不需要分析

### 操作步骤
1. 打开飞书私密收集群
2. 滚动查看所有评估卡片
3. 每条消息包含：
   - 被评估人信息
   - 6个维度评分
   - 总分和评级
   - 评估编号和提交时间

---

## 方法2：导出飞书聊天记录

### 适用场景
- 需要永久保存
- 需要离线查看

### 操作步骤

#### 方式A：在桌面端导出
1. **打开飞书桌面客户端**
2. **进入私密收集群**
3. **点击右上角** "..." → **更多** → **导出聊天记录**
4. **选择时间范围**：评估活动的开始和结束日期
5. **选择导出格式**：
   - TXT（纯文本，简单）
   - PDF（格式化，适合存档）
6. **点击导出** → 选择保存位置
7. 导出的文件包含所有评估消息的文本内容

#### 方式B：复制消息内容
1. 在飞书群中找到每条评估消息
2. 长按消息 → **复制**
3. 粘贴到Excel或Word中
4. 重复操作所有评估消息

---

## 方法3：使用收集工具汇总（推荐）

### 前提条件
已在GitHub Pages部署了系统，或本地有HTML文件

### 操作步骤

1. **打开数据收集工具**
   ```
   https://你的用户名.github.io/仓库名/assessment-collect.html
   ```
   或本地打开：`assessment-collect.html`

2. **准备数据**

   **选项A：如果评估人直接发JSON文件给您**
   - 收集所有人发给您的JSON文件
   - 放在同一个文件夹

   **选项B：从飞书群手动整理**
   - 从飞书消息中复制评估数据
   - 整理成JSON格式（见下方格式说明）

3. **导入数据到收集工具**
   - 点击"添加评估数据"按钮
   - 方式1：直接粘贴JSON内容
   - 方式2：拖拽JSON文件到页面
   - 方式3：点击"选择文件"上传

4. **查看统计和分析**
   - 自动生成统计图表
   - 按部门、维度分析
   - 查看评级分布

5. **导出汇总数据**
   - 点击"导出Excel"按钮
   - 或点击"导出CSV"按钮
   - 或点击"导出JSON"保存原始数据

## 方法4：手动整理到Excel

### 适用场景
- 需要自定义表格格式
- 需要添加额外分析

### 操作步骤

1. **创建Excel表格**，包含以下列：
   ```
   评估编号 | 被评估人 | 部门 | 职位 | 维度1 | 维度2 | 维度3 | 维度4 | 维度5 | 维度6 | 总分 | 评级 | 提交时间
   ```

2. **从飞书群复制数据**
   - 打开每条评估消息
   - 复制相关信息
   - 粘贴到Excel对应列

3. **使用Excel公式计算**
   ```
   总分：=SUM(E2:J2)
   百分比：=L2/30*100
   平均分：=AVERAGE(E2:J2)
   ```

4. **创建数据透视表**
   - 选择数据范围
   - 插入 → 数据透视表
   - 按部门、维度分析

5. **生成图表**
   - 选择数据
   - 插入 → 图表
   - 选择柱状图、雷达图等

## 方法5：批量导出（高级）

### 如果您有技术能力，可以编写脚本

#### Python脚本示例（从JSON文件批量导出到Excel）

```python
import json
import pandas as pd
from pathlib import Path

# 读取所有JSON文件
data_folder = Path("评估数据")
all_assessments = []

for json_file in data_folder.glob("*.json"):
    with open(json_file, 'r', encoding='utf-8') as f:
        data = json.load(f)
        all_assessments.append({
            '评估编号': data['assessmentId'],
            '被评估人': data['employeeName'],
            '部门': data['department'],
            '职位': data['position'],
            '维度1': data['scores'][0],
            '维度2': data['scores'][1],
            '维度3': data['scores'][2],
            '维度4': data['scores'][3],
            '维度5': data['scores'][4],
            '维度6': data['scores'][5],
            '总分': data['totalScore'],
            '评级': data['rating'],
            '提交时间': data['submittedAt']
        })

# 导出到Excel
df = pd.DataFrame(all_assessments)
df.to_excel('评估数据汇总.xlsx', index=False)
print(f"已导出 {len(all_assessments)} 份评估数据")
```

## 📊 数据分析建议

### 基础统计
- 提交率：已提交人数 ÷ 总人数
- 平均总分：所有评估的平均分
- 评级分布：卓越/优秀/符合预期/需改进的人数

### 维度分析
- 每个维度的平均分
- 得分最高/最低的维度
- 各维度的分数分布

### 部门对比
- 按部门统计平均分
- 部门间维度对比
- 找出优势和待改进领域

### 个人报告
- 为每个被评估人生成报告
- 包含6个维度得分
- 优势和改进建议

## ⚠️ 常见问题

### Q1: 飞书能否直接导出为Excel？
**A**: 飞书只能导出TXT或PDF格式的聊天记录，需要手动或使用工具转换为Excel。

### Q2: 评估数据会丢失吗？
**A**: 不会。数据存储在：
- 飞书群的聊天记录（永久保存）
- 评估人的浏览器localStorage（除非清除）

建议定期导出备份。

### Q3: 如果评估人数很多（>50人），怎么高效导出？
**A**: 推荐方案：
1. 使用`assessment-collect.html`工具
2. 或编写Python脚本批量处理
3. 或请IT部门协助开发API接口

### Q4: 可以按时间筛选评估数据吗？
**A**: 可以：
- 飞书导出时选择时间范围
- 在Excel中按"提交时间"列筛选
- 使用收集工具的筛选功能

### Q5: 如何保护导出数据的安全？
**A**:
- 导出文件加密（Excel密码保护）
- 存储在安全位置（公司内网）
- 限制访问权限
- 定期删除不需要的副本
