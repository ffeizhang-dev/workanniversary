## 📝 定时代码

### JavaScript定时（浏览器/Node.js）

```javascript
// 每天早上9点检查
function scheduleDailyCheck() {
    const now = new Date();
    const target = new Date();
    target.setHours(9, 0, 0, 0);  // 设置为9点

    if (now > target) {
        target.setDate(target.getDate() + 1);
    }

    const timeUntilTarget = target - now;

    setTimeout(() => {
        blessingSystem.sendConfirmationToHR();
        scheduleDailyCheck(); // 递归设置下一次
    }, timeUntilTarget);
}

// 启动定时任务
scheduleDailyCheck();
```

### Cron任务（服务器）

```bash
# 每天早上9点执行
0 9 * * * node /path/to/anniversary-check.js
```

**说明**：
- `0` - 分钟（9:00的第0分钟）
- `9` - 小时（上午9点）
- `* * *` - 每天、每月、每周

## 💡 温馨提示

### 建议时间选择

**早上8点**：
- 适合：早起型团队，希望员工一上班就收到祝福
- 缺点：可能错过确认，太早HR未到岗

**早上9点**（当前设置）：✅
- 适合：大多数公司，HR有时间确认
- 优点：平衡了及时性和操作便利性

**早上10点**：
- 适合：希望HR有充足时间处理其他事务
- 缺点：可能略晚，员工已经开始工作

**中午12点**：
- 适合：午休时发送，轻松氛围
- 缺点：员工可能外出午餐错过
